# [FreeCAD][a]

!!! tip
    Always calculate parametric values in the sheet; values calculated in the
    sketch will be resolved at creation however the formula will not be stored.

!!! example "Preferences"
    * General:
        * Units:
            * Unit System: **Standard (mm/kg/s/degree)**
        * Output Window:
            * Show report view on error: ✘
    * Display ➔ 3D View:
        * Anti-Aliasing: **MSAA 8x**
    * Addon Manager ➔ Addon Manager options:
        * Automatically check for updates at start (requires GPython): ✔
        * Download Macro metadata (approximately 10MB): ✔

!!! example "Tools ➔ Edit parameters ➔ BaseApp ➔ Preferences ➔ Mod ➔ PartDesign"
    * [SwitchToTask][b]: ✘


Install **freecad.gears** workbench via the Addon manager. Requires restart.
