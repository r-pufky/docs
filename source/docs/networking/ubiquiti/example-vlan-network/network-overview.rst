.. _network-overview:

Network Overview
################
See :ref:`vlan-101` for detailed VLAN concept information. The example network
:ref:`example-network-diagram` will be laid out as follows:

+------------+----------------+
| Color      | Network        |
+============+================+
| White      | Trunk: All     |
+------------+----------------+
| Purple     | Local Router   |
+------------+----------------+
| Teal       | Trunk: Wired   |
+------------+----------------+
| Light Blue | Trunk: Server  |
+------------+----------------+
| Dark Blue  | Trunk: Wifi    |
+------------+----------------+
| Orange     | Wired          |
+------------+----------------+
| Green      | Server         |
+------------+----------------+
| Yellow     | Infrastructure |
+------------+----------------+
