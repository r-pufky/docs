Game Mode
#########
.. regedit:: Enable Game Mode
  :path:     HKEY_CURRENT_USER\Software\Microsoft\GameBar
  :value0:   AllowAutoGameMode, {DWORD}, 1
  :value1:   AutoGameModeEnabled, {DWORD}, 1
  :ref:      https://www.tenforums.com/tutorials/75936-turn-off-game-mode-windows-10-a.html
  :update:   2021-02-19
  :open:

  Disables Windows updates while playing games. Game "performance" optimizes are
  minimal.
